## Install packages

<p className={'_fs-400 _mxw-80ch _lh-165pct'}>
	First, install the required packages for react version of Nomimono.
</p>

- `@nomimono/nomimono-react` main package of Nomimono react components
- `@nomimono/nomimono-theme` theme package

<p className={'_fs-400 _mxw-80ch _lh-165pct'}>
	Open a terminal at the root of your project workspace and run the following command:
</p>

<Source language='bash'
        dark
        format={false}
        code={dedent`
            yarn add @nomimono/nomimono-react @nomimono/nomimono-theme
            # or if you use npm
            npm install @nomimono/nomimono-react @nomimono/nomimono-theme
        `}
/>

<p className={'_fs-400 _mxw-80ch _lh-165pct'}>
	The command will install all packages in the node_modules folder of your application.
</p>

<br/>

## Import CSS

<p className={'_fs-400 _mxw-80ch _lh-165pct'}>
	Import required CSS files into your project.
</p>

<Source language='jsx'
        dark
        format={false}
        code={dedent`
            @import '@nomimono/nomimono-css/reset.css'; // reset browser default style (optional)
            @import '@nomimono/nomimono-theme/default.css'; // default 'Nom Yen' theme
            @import '@nomimono/nomimono-react/style.css'; // component CSS
            ...
        `}
/>

<p className={'_fs-400 _mxw-80ch _lh-165pct'}>
	To use reset.css you need to install core CSS package first by typing <code>npm install @nomimono/nomimono-css</code> in your terminal.
</p>
